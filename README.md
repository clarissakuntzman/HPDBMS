This project is a web-based Healthcare Practice Database Management System designed for small medical facilities such as doctor’s offices and dental practices. Many small clinics still rely on paper records or outdated systems, which can lead to inefficiencies, scheduling errors, and data security risks. This system aims to improve workflow by centralizing patient and appointment data into a secure, easy-to-use digital platform.    
Accomplishments:  
Implemented secure user authentication with role-based access control (Admin vs Staff)  
Designed a relational database structure linking patients and appointments using patient IDs  
Created full CRUD functionality for patients and appointments (create, read, update, delete)   
Improved data integrity by replacing name-based scheduling with patient ID–based relationships   
Developed a structured multi-page UI with separate pages for patients, appointments, and analytics   
Built an analytics dashboard displaying patient and appointment statistics   
Added a calendar-based appointment view using FullCalendar.js   
Designed a modern, responsive UI with custom CSS including navigation bars, tables, and forms  
Enhanced user experience with separate add/edit pages for cleaner workflow   
Strengthened system usability with error handling and styled feedback messages   
Demonstrated practical application of full-stack development and database design principles
